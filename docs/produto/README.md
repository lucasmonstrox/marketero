# Produto

Responde **por quê · o quê · pra quem**. É o ponto de partida de leitura: a tese do
Marketero, os pilares e o anti-escopo. Quase todo outro documento referencia daqui.

**Entra aqui** o que é fundador (a visão, os pilares, o estado do produto) e não muda de
forma de uma semana pra outra.

**Não entra** investigação exploratória (vai pra [`investigacoes/`](../investigacoes/))
nem design de feature (vai pra [`design/`](../design/)).
